# HandPoseBarracuda_GestureRecognition
Mediapipe, Unity-Barracuda, ONNX を使ってUnity上でての形状認識を実装したもの

このリポジトリはkeijiro氏の[HandPoseBarracuda](https://github.com/keijiro/HandPoseBarracuda)を基にして独自のONNXファイルによるジェスチャの推論を組み込んだものです。
